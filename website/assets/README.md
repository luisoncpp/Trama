# Help screenshot assets

PNG files in this folder are generated by:

```bash
npm run help:screenshots
```

The command builds the app, boots Electron in capture mode, opens `example-fantasy/`, drives fixed UI scenarios, and writes one PNG per scenario.

Re-run after sidebar, editor, map, or revision-rail chrome changes. Commit updated PNGs with the help HTML that references them.

The git-snapshots scenario may run `git init` in `example-fantasy/` when no repository exists. That folder is gitignored (`example-fantasy/.git`).
