# Relevant Markdowns for the Task

## Task

One broken image breaks all the project — it cannot be opened if it has a single broken image.

Requirements:
1. The project should be able to be opened even if there is a broken image.
2. A broken image must be rendered like the 🖼️ emoji.
3. Saving a document with a broken image should not change the image.
