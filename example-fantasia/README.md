Example: Proyecto de fantasía ligero — "Las Llamas del Lirio"

Este proyecto de ejemplo contiene:
- `book/`: 2 actos, 3 capítulos por acto, una escena por capítulo (texto narrativo).
- `outline/`: planes y beats por capítulo para guiar la escritura.
- `lore/`: fichas de personajes, lugares y sistemas (magia y política).

Tono: fantasía ligera, accesible, con referencias claras para usar como demostración.
