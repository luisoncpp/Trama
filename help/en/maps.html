Visual Maps & Markers

Trama supports visual maps as special documents under the lore/ directory. You can upload custom images (like fantasy worlds, building maps, or timeline charts) and place interactive pins/markers directly on them to link to other project lore files.

How to Access

To create a Map document:

  1. Open the Lore tab in the sidebar.
  2. Click the ▼ toggle on the split + Article button and choose Create map.
  3. Enter a map name and select an image file from your system to use as the map background.

Example Workflow

Suppose you are building a fantasy continent:

  1. Create a Map document named "Continent Map" with your world map image.
  2. Double-click anywhere on the map background to place a pin/marker.
  3. Right-click the pin to link it to a location lore article (e.g., lore/cities/royal-capital.md).
  4. Clicking the pin will open the linked document side-by-side or in the active pane.
