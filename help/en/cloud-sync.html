Cloud Synchronization

Since Trama is a file-first application, your project data lives in simple directories on your computer. This makes it very easy to keep your work backed up and synchronized across multiple devices.

Recommended Cloud Providers

To synchronize your project automatically across computers, we suggest placing your Trama project folder inside a cloud storage directory. Some great options include:

  • Google Drive
  • OneDrive

These services will automatically detect changes as Trama saves your markdown files and sync them to the cloud.

Advanced Control with Git

If you need more control over your versions, branching, or collaboration with others, we strongly recommend using a local Git repository. By installing Git locally, you can use Trama's built-in Git Snapshots feature to manage your revisions manually, offering robust history tracking on top of any cloud sync solution.
