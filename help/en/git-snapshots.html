Git Snapshots & Revision Rail

Trama integrates with Git to provide local revision tracking. You can take snapshot saves of your project, browse past revisions of any file, and view side-by-side visual diffs before restoring older text states.

How to Access

  • See Revisions: Right-click in the editor and choose See Revisions from the context menu. The revision rail opens on the right for the active document.
  • Save Snapshot: Open the Import and export sidebar panel and click Save Snapshot under Project history to commit Trama-managed project content with a message.

Example Workflow

To restore text you deleted yesterday:

  1. Right-click in the editor of the chapter and select See Revisions. The revision list displays on the right side.
  2. Click a revision from the list. Trama displays a read-only side-by-side comparison showing additions and deletions.
  3. Click Restore revision in the editor toolbar to write that past version back to your active document.
